Attributes for form: • accept-charset : we use this attribute to encode our data form • autocomplete : we use this attribute to auto fill input element by browser • name : Name of the element. server uses the name attribute to identify the fields in form submits. • rel : This attribute specifies the relationship between the link and the target • action : this attribute show that where our data(form) send .Our data is processed at this address • enctype : When sending information through the form, it encrypts the data • method : This attribute receives or sends data form, from an address • novalidate : this attribute shows that the form shouldn't be validated when submitted • target : Indicates where to display the response after submitting the form
